# 20.4.04 Interactive Art
```
20.4.04 Interactive Art
```

You need to create multiple interactive objects in your project that use UI elements (sliders, radio buttons, checkboxes) to change your artwork. 

- Changing sizes
- Changing rotations
- Using a `for` loop to create multiple items

---

### **Part 1: Basic Interactive Art**  
- **Switch Between Color Modes**: Allow the user to switch between `RGB`, `HSB`, `HSL` color modes
  - See the refference for `colorMode` 
- **Checkbox**: Use at least one checkbox that can toggle a feature on or off.
  - Example: Show or hide an effect, or toggle the background color.
- **Radio Button Group**: Use a radio button group with three or more transparency options.
   Reminder: Transparency is also called `alpha`, and it’s the fourth argument in `fill(r, g, b, alpha)` ranges 0-255

---

### **Part 2: User Input Interaction**
- **Additional Checkbox**: Add another checkbox to control an additional feature of your choice
- **Slider**: Use a slider that controls the number of shapes drawn in a `for` loop.
- **Push Button**: Use a push button that triggers a window alert with a pun when clicked.
- **Scrolling Text**: Have a text string that scrolls across the top of the screen. Use a slider to control the speed of the scrolling text.
- **Scaling**: Use a slider to control the `scale()` function, affecting the overall size of the shapes in your artwork.
